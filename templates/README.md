# Templates

Reference implementations for apps and integrations. Browse these for patterns, or ask your AI to set one up — it will read the template and guide you through configuration.

## Apps

Example automations demonstrating common patterns. Copy one to `apps/` to get started.

| Template | Pattern | Purpose |
|----------|---------|---------|
| [goals](apps/goals/) | Reference | Personal and professional goal tracking |
| [health](apps/health/) | Data Sync | Health biomarker tracking with staleness alerts |
| [news-digest](apps/news-digest/) | Timer + Email | AI-curated daily news digests |
| [todays-agenda](apps/todays-agenda/) | Briefing | Morning briefing with calendar, tasks, and goals |
| [error-remediation](apps/error-remediation/) | System/Event | Auto-recovery for failed automation workflows |
| [project-index](apps/project-index/) | System | Generates projects/README.md from frontmatter |

## Integrations

Connectors for external tools and services. Copy one to `integrations/` and configure it.

See [integrations/README.md](../integrations/README.md) for the full categorized list.
